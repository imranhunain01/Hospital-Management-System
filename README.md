## Overview

The Hospital Management System is a desktop application developed using C# Windows Forms to simplify hospital operations. The system provides an easy-to-use interface for managing patients, doctors, appointments, and other administrative tasks in a centralized environment.

This project was developed as a personal learning project to enhance practical skills in desktop application development and database management.

---

## Features

- User Registration
- Secure Login System
- Dashboard
- Patient Management
- Doctor Management
- Appointment Scheduling
- Database Integration
- User-Friendly Interface

## Installation

1. Clone the repository.

```bash
git clone https://github.com/imranhunain01/Hospital-Management-System.git
```

2. Open the solution file in Visual Studio.

3. Restore the required NuGet packages (if prompted).

4. Configure the SQL Server LocalDB connection.

5. Execute the database script (`HospitalDB.sql`).

6. Build and run the project.
